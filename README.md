# TeleMaster Pro - Sistema de Transmisión Profesional

Sistema de transmisión profesional para televisión, streaming e IPTV.

## Características
- Interfaz web moderna y profesional
- Soporte para hardware profesional (Blackmagic, AJA, NDI)
- Transmisión multidestino simultáneo
- Sistema IPTV integrado
- Multivisor personalizable
- Grabación en múltiples formatos

## Instalación Rápida

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/teleMaster-pro.git
cd teleMaster-pro

# Instalar dependencias
npm install

# Iniciar servidor
npm start
