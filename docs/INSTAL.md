# Guía de Instalación - TeleMaster Pro

## Requisitos del Sistema

### Hardware Mínimo:
- CPU: Intel i5 o AMD Ryzen 5 (4+ cores)
- RAM: 16 GB
- Almacenamiento: 100 GB SSD
- GPU: NVIDIA GTX 1050 o equivalente
- Tarjeta de captura: Blackmagic DeckLink, AJA, o compatible

### Hardware Recomendado:
- CPU: Intel i7/i9 o AMD Ryzen 7/9 (8+ cores)
- RAM: 32 GB o más
- Almacenamiento: NVMe SSD 500 GB+
- GPU: NVIDIA RTX 3060 o superior
- Tarjeta de captura: Blackmagic DeckLink 8K Pro

### Sistema Operativo:
- Ubuntu 20.04 LTS o superior
- Windows 10/11 (64-bit)
- macOS 12+ (solo desarrollo)
